Semiconductor Properties Prediction

Python

In this project, I use machine learning to optimize the transport integral of tetracene. Organic semiconductors are responsible for the OLED panels found on high end TV's and cell phone screens. Computing electronic properties of theoretical materials, such as the molecular transport integral, is computationally expensive. Using machine learning, we can predict (and optimize) the transport integral of a material. Given the coulomb interactions (found as a function of the structure), we predict the transport integrals. My predictions were 93% accurate when tested against the unseen professor's test data for final grading. PCA was critical to reduce the computational cost, without affecting performance. The best method for the predictions was a neural net made using Keras, though Kernal Ridge performed quite well. All tests were run with k-fold cross-validation.
